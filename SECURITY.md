# Security Policy

## Supported versions

Only the latest tagged minor release is supported with security fixes while
the project is at `0.x`. Once `1.0.0` is cut this section will be expanded.

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Reporting a vulnerability

Please **do not** file a public issue. Instead, open a private
[security advisory](https://github.com/smoreg/bf/security/advisories/new) on
GitHub. Include:

- Affected version or commit
- A reproduction or proof of concept
- Impact assessment in your own words

You can expect an acknowledgement within 7 days. If a fix is needed it will
be released as a patch version and credited to you in the changelog (unless
you prefer to remain anonymous).
